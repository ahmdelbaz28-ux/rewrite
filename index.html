How to Use – 10 Ways

  1. CLI – run smartlangguard fix "text" directly in terminal.
  2. Interactive mode – smartlangguard interactive for a chat‑like session.
  3. Detect command – smartlangguard detect "text" to see mistakes without fixing.
  4. File fix – smartlangguard fix -f input.txt -o output.txt for batch processing.
  5. Pipe usage – echo "text" | smartlangguard fix for quick one‑liners.
  6. Daemon – run smartlangguard daemon to monitor clipboard and use the global hotkey.
  7. Browser extension – after the daemon is running, use the Chrome/Edge extension to fix selections.
  8. VS Code extension – install the extension to fix selections or the last typed word.
  9. MCP server – expose the tool to AI assistants (Claude, Cursor, etc.).
  10. Node.js API – programmatically call core.fixText() in your own applications.
