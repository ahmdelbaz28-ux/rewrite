Key Features

  • 🔧 CLI tool – works on Windows, macOS and Linux.
  • 🖥️ Background daemon with global hotkey.
  • 🌐 Browser extension (Chrome MV3).
  • 🧩 VS Code extension – fix on the fly.
  • ⚡ Real‑time typing detection with sound alerts.
  • 🔐 Secure license validation and telemetry.

Installation

Install globally via npm:

npm install -g @smartlangguard/cli

Or download a pre‑built binary for your platform:

  • Windows x64
  • macOS Intel
  • Linux x64

Verify the checksums.sha256 file in the release assets for integrity.
