SmartLangGuard

تصحيح تلقائي وفوري لأخطاء تخطيط لوحة المفاتيح بين العربية والإنجليزية. يعمل في كل التطبيقات — Word, Chrome, VS Code, Telegram والمزيد.

⌨️

شاهد SmartLangGuard في العمل

اكتب "high hofhv;" وسيتحول تلقائياً إلى "اهلا اخبارك" 🎯
يعمل في Word، Chrome، VS Code وكل تطبيقاتك المفضلة.

npm install -g @smartlangguard/cli
echo "high hofhv;" | smartlangguard fix

⚡ فوري <1ms للترجمة 🖥️ عبر المنصات Windows / macOS / Linux 🔌 بدون إنترنت يعمل Offline

✨ المميزات

🔄

تصحيح فوري على مستوى النظام

باستخدام WH_KEYBOARD_LL Hook في Windows، يعترض ضغطات المفاتيح ويصحح الأخطاء فوراً في أي تطبيق.

🧠

محرك ترجمة ذكي

محرك قواعد + تقييم بالذكاء الاصطناعي للحالات الغامضة. يدعم التصحيح في الاتجاهين: عربي → إنجليزي والعكس.

🪟

System Tray App

تطبيق Electron صغير يعيش في شريط المهام. أيقونة خضراء = شغال، حمراء = متوقف. toggle للتفعيل/التعطيل.

⌨️

يدعم AZERTY / QWERTZ

يكشف تخطيط لوحة المفاتيح تلقائياً (QWERTY, AZERTY, QWERTZ) باستخدام Win32 API ويضبط الترجمة وفقاً لذلك.

🔗

MCP Protocol

يدعم Model Context Protocol للتكامل مع Claude Desktop, Cursor, Cline وأدوات AI الأخرى.

🔐

خصوصية كاملة

التعامل مع النصوص محلياً 100%. لا بيانات تغادر جهازك. يعمل بدون إنترنت. تقارير الأداء اختيارية وخفيفة.

🚀 التثبيت

اختر طريقة التثبيت المناسبة لك

# تثبيت عالمي عبر npm npm install -g @smartlangguard/cli # التحقق من التثبيت smartlangguard --version 0.3.0 # تصحيح سريع echo "high hofhv;" | smartlangguard fix
# Windows — تحميل الملف الثنائي curl -L -o smartlangguard.exe https://github.com/ahmdelbaz28-ux/rewrite/releases/latest/download/smartlangguard-win-x64.exe # macOS (Apple Silicon) curl -L -o smartlangguard https://github.com/ahmdelbaz28-ux/rewrite/releases/latest/download/smartlangguard-macos-arm64 chmod +x smartlangguard sudo mv smartlangguard /usr/local/bin/ # Linux curl -L -o smartlangguard https://github.com/ahmdelbaz28-ux/rewrite/releases/latest/download/smartlangguard-linux-x64 chmod +x smartlangguard sudo mv smartlangguard /usr/local/bin/
# تشغيل الخلفية مع مراقبة لوحة المفاتيح (Windows) npx @smartlangguard/daemon --enable-typing-monitor # أو تثبيت دائم وتشغيل npm install -g @smartlangguard/daemon smartlangguard-daemon --enable-typing-monitor --auto-start # يظهر notification: ✓ Fixed: high → اهلا

🛠️ التقنيات المستخدمة

⚡

Node.js + C#

المحرك الأساسي في Node.js (قواعد + تقييم). Hook النظام في Windows باستخدام C# و Win32 API.

🖼️

Electron Tray

تطبيق شريط المهام بأيقونات ديناميكية مولّدة برمجياً بدون ملفات صور خارجية.

🔄

npm Workspaces

Monorepo بإدارة npm workspaces: core, cli, daemon, mcp-server, windows-hook, tray-app.

📦

pkg للتوزيع

بناء ملفات ثنائية (binaries) عبر @yao-pkg/pkg مع دعم Windows, macOS, Linux.
