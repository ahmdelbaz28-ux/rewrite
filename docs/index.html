SmartLangGuard

Fix Arabic/English keyboard layout mistakes across CLI, Daemon, Browser and VS Code.

Install

Install the CLI globally via npm:

npm install -g @smartlangguard/cli

Or download a pre‑built binary for your platform:

  • Windows x64
  • macOS Intel
  • Linux x64

Verify checksums (SHA‑256) are present in the release assets.
